# ---- Định danh & Thời gian ----
# Xác định bối cảnh của dữ liệu.
- run_date: Ngày chạy dữ liệu
- retailer_id: Id gian hàng
- branch_id: Tên chi nhánh của gian hàng
- timeframe_start: Ngày bắt đầu chu kỳ
- timeframe_end: Ngày kết thúc chu kỳ


# ---- Phân loại Khách hàng theo RFM ----
# Dữ liệu dùng cho biểu đồ Treemap và các biểu đồ thanh Doanh thu, Trả hàng, Lợi nhuận gộp theo nhóm RFM.
- grouping_customer_rfm: 1 array nhóm khách hàng rfm(1- Khách hàng Trung thành, 2- Khách hàng Thân thiết, 3- Khách hàng Tiềm năng, 4- Khách hàng Cần quan tâm, 5- Khách hàng Sắp rời bỏ.)
- num_customer_rfm: 1 array chứa số lượng khách hàng tương ứng với nhóm khách
- customer_ratio_rfm: 1 array chứa tỷ lệ khách hàng tương ứng với nhóm khách
- revenue_rfm: 1 array chứa doanh thu tương ứng với nhóm khách
- return_value_rfm: 1 array chứa giá trị trả hàng tương ứng với nhóm khách
- profit_rfm: 1 array chứa lợi nhuận gộp tương ứng với nhóm khách
